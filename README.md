# vue-deploy-webpack

## Project setup

```
npm install
```

Add Stylus Mixins
```
cd src
git clone https://github.com/Shanery/Stylus-Mixins.git
mv Stylus-Mixins styles
cd ..
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
